# Springboot-Liquibase-app
Liquibased application, dynamically handle this application

How can you use this application

        1. Clone this repo
            ->  git clone https://github.com/MrAkashKumar/springboot-liquibase-app.git

        2. Import into IDE
        
        3. RUN Command - mvn clean install 

        4. Run this application 

        5. Open a browser and paste this url - http://localhost:7082/h2-console

        6. Check Test connect and connect 

        7. you can check - Table automatcally has created !
